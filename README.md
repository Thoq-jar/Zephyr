# Zephyr

**PLEASE FOLLOW SETUP GUIDE BEFORE DOING ANYTHING ELSE!**

## Setup
- Make an env:
```bash
python3 -m venv .venv
```

- Enter the env:
```bash
. .venv/bin/activate
```

- Install dependencies:
```bash
pip install -r requirements.txt
```

---
## Building
- Run build script:
```bash
python utils/build.py
```

## Running
- Run main script:
```bash
python main.py
```

## Installing
- Build Zephyr with the Building writeup
- Run 
```bash
sudo ./utils/install.py
```
